This is a demo JPublish web application using the new StringTemplate Component for
view rendering.

Learn more about the StringTemplate library:
http://www.antlr.org/wiki/display/ST/StringTemplate+Wiki+Home
http://www.cs.usfca.edu/~parrt/course/601/lectures/stringtemplate.html

Two libraries are required by this demo to run: stringtemplate.jar and and
antlr-2.7.7.jar, both distributed with the demo's WEB-INF/lib folder.
